3.1

Command Line은  cmd,프롬프트 등으로 불리는 텍스트기반의 응용프로그램이다.

\$(리눅스,맥), >(윈도우) 접두어 이다.

현재 디렉터리 확인법 :pwd(맥), cd(윈도우)를 입력하여 확인한다.

파일과 디렉터리 목록보기: ls(맥),dir(윈도우)를 입력하여 디렉터리에 어떤것이 있는지 확인한다.

현재 디렉터리 변경: 맥이랑 윈도우가 동일하게 cd 원하는 디렉터리를 입력하면 변경 할 수 있다.

새 디렉터리 생성/삭제 : 맥이랑 윈도우가 동일하게 mkdir 원하는이름을 입력하여 생성 할 수 있다.

그 후 rmdir을 이용하여 삭제 가능하다. 하지만 복구 불가능하니 주의 해야한다.
